# ⚡ Comandos Rápidos para Crear Ejecutable .exe

## Instalación
pip install pyinstaller

## Construcción (Recomendado)
python build_exe.py

## Ubicación del .exe
dist/IT-Ops-CLI.exe

## Probar el ejecutable
dist/IT-Ops-CLI.exe

## Notas
- Tamaño aproximado: 50-100 MB
- Requiere Ansible instalado en el sistema destino
- El código está protegido como bytecode compilado
